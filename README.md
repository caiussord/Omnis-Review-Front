Tecnologias Utilizadas
Framework: Vue.js 3 (Composition API)

Linguagem: TypeScript

Estilização: Tailwind CSS

Build Tool: Vite

Gerenciamento de Estado: Pinia (planejado)

Roteamento: Vue Router

Proposta Visual
O design adota uma abordagem Dark Modern, utilizando uma paleta de cores baseada em tons de roxo e azul profundo.

Estética: Uso de Glassmorphism (efeitos de desfoque e transparência) em cards e formulários.

Componentização: Inputs e botões personalizados com transições suaves e feedbacks visuais de estado (hover, active, focus e loading).

Cores: Gradientes entre Violet (#8b5cf6) e Blue (#3b82f6) sobre fundos Slate (#0f172a).

Pré-requisitos
Antes de começar, você precisará ter instalado em sua máquina:

Node.js (versão 18 ou superior)

Gerenciador de pacotes NPM ou Yarn

Instalação e Execução
Clone o repositório:

Bash
git clone https://github.com/caiussordillo/omnis-review-front.git
Entre no diretório do projeto:

Bash
cd omnis-review-front
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
